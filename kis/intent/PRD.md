# Intent — product

## Vision
Give one job-seeker an AI "command center" that runs on their own machine and their existing Claude Max subscription — no extra API bill — to cut the busywork of searching, tailoring, and tracking applications.

## Goals
- Find fresh, relevant jobs from open sources (remote boards + Greenhouse/Lever/Ashby).
- Score each against the user's resume with an honest match score + one-line reason.
- Draft a cover letter and a short recruiter outreach email per job.
- Auto-tailor an ATS-safe CV for weaker matches (score < 75), surfacing genuine gaps honestly — never fabricating skills.
- Track the pipeline: New → Applied → Screening → Interview → Offer (+ Rejected).

## Non-goals / guardrails
- No auto-apply. The user reviews and submits.
- No fabrication in tailored CVs or scoring (product/ethics rule, enforced in prompts).
- Not a shareable web service — local, single-user only.

## Status
Feature-complete and running. No active feature work requested.
